# ⚫⚪ AI 바둑 게임 서버 운영 가이드

이 가이드는 Rocky Linux(또는 유사한 Linux 환경)에서 스프링 부트 서버를 실행하고 종료하는 방법을 설명합니다.

---

## 🚀 1. 서버 실행하기 (백그라운드 실행)

SSH 세션이 끊겨도 서버가 계속 작동하도록 `nohup`을 사용하여 실행합니다. 현재 포트는 **7778**을 사용하도록 설정되어 있습니다.

### 실행 명령어
```bash
nohup java -jar ai-go-0.0.1-SNAPSHOT.jar --server.port=7778 > go.log 2>&1 &
```

### 명령어 상세 설명
- `nohup`: 터미널 세션이 종료되어도 프로그램을 중단하지 않음.
- `> go.log`: 모든 로그 기록을 `go.log` 파일에 저장.
- `2>&1`: 에러 메시지도 로그 파일에 함께 기록.
- `&`: 백그라운드에서 실행.

---

## 🛑 2. 서버 종료하기

실행 중인 서버를 종료하는 방법은 두 가지가 있습니다.

### 방법 A: 간단한 종료 (추천)
프로세스 이름으로 한 번에 종료합니다.
```bash
pkill -f ai-go
```

### 방법 B: 수동 종료 (PID 찾기)
1. **프로세스 번호(PID) 확인**
   ```bash
   ps -ef | grep ai-go
   ```
2. **출력된 결과의 두 번째 숫자를 확인하여 종료** (예: 번호가 1234인 경우)
   ```bash
   kill -9 1234
   ```

---

## 🔍 3. 상태 확인 및 관리

### 포트 확인 (7778 포트가 열려 있는지)
```bash
netstat -nlp | grep :7778
```

### 로그 실시간 확인
서버가 잘 작동하는지, 에러는 없는지 실시간으로 확인하고 싶을 때 사용합니다.
```bash
tail -f go.log
```
*(중단하려면 `Ctrl + C`를 누르세요. 서버는 종료되지 않습니다.)*

---

## ⚠️ 주의사항
- 서버 실행 시 반드시 **Java 17 이상** 버전이 필요합니다.
- DB 연결 정보는 `src/main/resources/application-local.yml` 설정을 따릅니다.
- 포트 번호를 변경하고 싶다면 실행 명령어의 `--server.port=7778` 부분을 수정하세요.
